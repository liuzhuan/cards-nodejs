# Node.js

来源：[Node.js 官网](https://nodejs.org/en/)

Node.js&reg;是基于 [Chrome V8 引擎](https://developers.google.com/v8/)的 JavaScript 运行时环境。Node.js 使用事件驱动的，非阻塞 I/O 模型，轻量高效。Node.js 的包管理工具 [npm](https://www.npmjs.com/), 是世上最大开源生态圈。
